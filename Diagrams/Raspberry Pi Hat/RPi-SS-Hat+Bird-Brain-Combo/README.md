This is a 4-layer version of the RPi Hat by [FireflySE](https://github.com/FireflySE) of Sweden, with the LIDAR Hydrometer Reader built-in and without the stepper motor driver _(Boilermaker only)_.

I personally have not tried this board yet, but you will most definitely need to use shielded cables for **everything** related to the hydrometer, especially the flow sensor. Keep in mind that the flow sensor is a capactitor and the longer the wires to it, the larger the capacitor becomes.

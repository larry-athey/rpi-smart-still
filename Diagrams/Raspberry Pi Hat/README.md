# Important Note

While each of these RPi Hat printed circuit boards has a footprint for a 5 volt 5 amp buck regulator, you will likely find that every one of them that size can only run a Raspberry Pi Model 3B+.

If you are going to use anything newer or a higher performance Banana Pi or Orange Pi, I am going to recommend that you use an external 12 volt to 5 volt regulator, such as those that you would use in an automotive setting. For example [this one](https://www.amazon.com/dp/B0FYNCSV2Z) as shown below.

The smaller ones that fit in the footprint on the printed circuit boards will go into overload shutdown and will stay latched in shutdown until all power has discharged. I realize these are bulkier, but the little ones just can't handle the current draw of today's single board computers.

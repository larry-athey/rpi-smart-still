# Important Note

While each of these RPi Hat printed circuit boards has a footprint for a 5 volt 5 amp buck regulator, you will likely find that every one of them that size can only run a Raspberry Pi Model 3B+.

If you are going to use anything newer or a higher performance Banana Pi or Orange Pi, I am going to recommend that you use an external 12 volt to 5 volt regulator, such as those that you would use in an automotive setting. Such as [this one](https://www.amazon.com/dp/B0FYNCSV2Z) as shown below.
